## LSTM

LSTM (Long Short Term Memory)는 forget gate, input gate, output gate를 이용해 이전 시점의 hidden state를 현시점에 반영하는 알고리즘으로, 구조는 다음과 같다.

LSTM은 주로 자연어 처리에 사용되는데, 주어진 데이터를 토대로 직접 코드를 작성하며 실습해보았다.
